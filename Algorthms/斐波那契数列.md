###  菲波那契数列  
在fib函数中填入代码，返回菲波那契数列的第n个数的值。 菲波那契数列是指每一位都是前两位相加之和的一个数列，一个有限的菲波那契数列如下所示  
0 1 1 2 3 5 8 13 ...  
举例来说  
-  fib(0)的返回值是 0  
-  fib(1)的返回值是 1  
-  fib(2)的返回值是 1( 因为1 = 0 + 1 )  
-  fib(3)的返回值是 2( 因为2 = 1 + 1 )  
-  fib(4)的返回值是 3( 因为3 = 2 + 1 )  
```js
function fib(n){
  var arr=[];
  arr[0]=0;
  arr[1]=1;
   for(var i=1;i<n;i++)
   {
      arr[i+1]=arr[i]+arr[i-1];
   }
   return arr[n];
}
```
